Errore nella generazione del report

Non è stato possibile generare il report di coverage completo. La cartella coverage non è stata creata.

Possibili soluzioni

  1. Verifica che la configurazione di Vitest sia corretta
  2. Assicurati che l'opzione coverage.all sia impostata su true
  3. Controlla che ci siano file sorgente nella cartella src
  4. Prova a eseguire manualmente il comando: npx vitest run --coverage

Configurazione attuale

coverage: {
  provider: "v8",
  reporter: ["text", "text-summary", "json", "html", "lcov", "cobertura"],
  reportsDirectory: "./coverage",
  all: true,
  include: ["src/**/*.{js,jsx,ts,tsx,vue}"],
  perFile: true,
  skipFull: false
}
